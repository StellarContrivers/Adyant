Salient Features

We at Adyant offer you an intuitive yet fun experience at learning Sanskrit.

Simple UI built for everyone

We offer personalised progress track record to help you stay focussed.

  • Ability to resume the lesson and save progress
  • Ability to select your level of choice like Novice, Beginner, Intermediate
  • Hints in questions to help you clear the section with ease

Pictorial Feedback

Pictures speak more than words. For novice learners we have picture based learning. It's scientifically proven that image aided learning helps in memory retention.

Many lessons have integrated images to help you understand the facts easily.

Reward Points (Dravinas)

Users gets reward to keep them motivated.

  • 5 Dravinas are awarded for each completed Section
  • 20 Dravinas are awarded for each completed Lesson
  • 1 Dravinas per section as Bonus for users who don't use hints

Special Ahna points to those who learn continuously for 10 days, 20 days and 30 days.

Reward points help people stay on track and maintains continuity among the learners. It promotes that they should visit and learn small sections regularly.

Trade-In Shop

You've got dravinas, we've got extra lessons. Trade in your earned Dravinas(Points) to unlock exciting lessons from our very own shop.

Audio Based Questions

Language is mastered not only by texts but also by speech and we aim to clear all obstacles on people's way towards excellence.

We've pronunciation audio for basic sanskrit alphabets.

How to install the software

Steps to install

1. Download and open the zip file

2. Python must be installed into the system.

3. 'pip install' all the python libraries mentioned below:

  • PyQt5
  • simpleaudio
  • indic_transliteration
  • webbrowser
  • csv
  • os

4. Run the Adyant.py script.

5. . Enjoy learning Sanskrit !!!
